# PACA++

**Provenance-Anchored Continuous Authorization Framework for Autonomous LLM Agents**

## Overview

PACA++ is a research framework for continuous authorization in autonomous AI agents.

It provides:

- Provenance-aware policy evaluation
- Continuous authorization
- Policy compilation
- Differential semantic verification
- Obligation enforcement
- Multi-engine policy validation

The framework is being developed as part of PhD research on secure autonomous AI systems.
